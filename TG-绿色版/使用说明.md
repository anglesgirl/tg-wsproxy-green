# Telegram 绿色版(内置 WSS 代理)

## 这是什么

一个零配置的 Telegram Desktop 绿色版,内置 WSS 代理隧道。

启动器自动拉起代理 → 用户首次点连接 → 之后自动启动 Telegram,关 TG 自动清理代理。

## 目录结构

```
TG-绿色版/
├── 启动Telegram.exe      ← 双击这个
├── 使用说明.md
├── Telegram/             ← 官方 Telegram Desktop v6.9.3
│   └── Telegram.exe
└── wsproxy/              ← tg-ws-proxy v1.8.1
    └── TgWsProxy.exe
```

## 使用方法

### 首次运行

1. 双击 `启动Telegram.exe`
2. 弹出提示框,点「确定」
3. 代理窗口打开,在里面点「连接」按钮
4. 连接成功后,点第二个提示框的「确定」
5. Telegram 自动启动,正常使用

### 以后运行

直接双击 `启动Telegram.exe`,代理自动启动 → 5秒后 Telegram 自动启动,无需任何操作。

关闭 Telegram 后,代理自动清理。

## 特点

- 独立 exe(Go 编译,1.8MB,无依赖)
- 完美支持中文路径
- 首次运行显示代理窗口,方便用户操作
- 后续运行代理窗口可见但不干扰
- 关 TG 自动清理代理进程
- 代理已运行时不重复启动

## 开机自启

右键 `启动Telegram.exe` → 创建快捷方式 → Win+R 输入 `shell:startup` → 拖进去
