# Telegram 绿色版(内置 WSS 代理)

## 这是什么

一个零配置的 Telegram Desktop 绿色版,内置 WSS 代理隧道。

**原理**:启动器(`启动Telegram.exe`)先拉起本地 wsproxy,再启动 Telegram。
Telegram 通过 SOCKS5 代理 `127.0.0.1:1080` 连接,流量被封装成 WSS 到 `kws*.web.telegram.org`,
效果等同于安卓 Nullgram 的内置 wsproxy。关闭 Telegram 后,启动器自动清理代理进程。

## 目录结构

```
TG-绿色版/
├── 启动Telegram.exe      ← 双击这个(独立程序,无依赖)
├── 使用说明.md
├── Telegram/             ← 官方 Telegram Desktop v6.9.3
│   └── Telegram.exe
└── wsproxy/              ← tg-ws-proxy v1.8.1
    └── TgWsProxy.exe
```

## 使用方法

1. 下载 zip,解压到任意位置(支持中文路径)
2. 双击 `启动Telegram.exe`
3. **首次**在 Telegram 里配置代理:
   - 设置 → 高级 → 连接类型 → 使用自定义代理
   - 添加代理 → SOCKS5
   - 服务器: `127.0.0.1`  端口: `1080`
   - 用户名/密码留空,保存
4. 以后每次双击 exe 即可,代理自动启停

## 特点

- **独立 exe 启动器**:Go 编译,2.3MB,无运行时依赖
- **完美支持中文路径**:不会出现脚本编码问题
- **静默后台运行**:不弹黑窗口
- **自动进程管理**:启动时拉起代理,关闭 TG 后自动清理
- **端口检测**:代理已在运行时不会重复启动
- **错误弹窗**:找不到文件或启动失败时有友好提示

## 开机自启

右键 `启动Telegram.exe` → 创建快捷方式 → Win+R 输入 `shell:startup` → 把快捷方式拖进去

## 常见问题

### Q: 启动后 Telegram 连不上?
检查 Telegram 代理设置是否为 SOCKS5 127.0.0.1:1080。
首次使用必须手动配置一次,之后自动保存。

### Q: 端口 1080 被占用?
编辑 `启动Telegram.exe` 源码(见 GitHub 仓库)修改端口,
同时修改 Telegram 代理设置里的端口。

### Q: 杀毒软件拦截?
tg-ws-proxy 开了本地监听端口,可能被误报,添加信任即可。
本方案不修改任何系统设置,纯绿色。

### Q: 可以放 U 盘用吗?
可以。整个目录绿色无依赖,放哪都能跑。
